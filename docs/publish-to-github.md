After creating an empty GitHub repository, run these commands from the repository folder:

```powershell

git add .
git commit -m "Create IT internship portfolio"
git branch -M main
git remote add origin https://github.com/rd4028/visualization-studio-portfolio.git
git push -u origin main
```
