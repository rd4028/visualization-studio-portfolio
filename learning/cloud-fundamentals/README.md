## EC2 Understanding

Amazon EC2 provides virtual servers that can be used to run applications, host services, or support testing environments. Key operational concepts include instance types, security groups, key pairs, operating systems, and starting or stopping resources responsibly.

## S3 Understanding

Amazon S3 provides object storage for files, backups, static assets, logs, and data. Key concepts include buckets, objects, permissions, versioning, storage classes, and public access controls.
