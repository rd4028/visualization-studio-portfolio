# Evidence Guide

This folder is for evidence collected while testing the mini project.

## Recommended Evidence

| Evidence | Purpose |
| --- | --- |
| Screenshot of the static site | Shows the deployed or locally rendered page |
| DNS command output | Shows the domain can be resolved |
| TCP port test output | Shows HTTPS connectivity on port 443 |
| HTTP status output | Shows the site responds successfully |
| Security checklist notes | Shows public sharing and sensitive-data checks were considered |
