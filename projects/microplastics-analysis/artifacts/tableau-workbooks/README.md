# Tableau Workbooks

This folder contains the Tableau packaged workbooks used as evidence for the completed Microplastics Analysis project.

## Files

| File | Original Purpose |
| --- | --- |
| [dataset-2-individual-part-question-1.twbx](dataset-2-individual-part-question-1.twbx) | Regional intervention and coastal contamination analysis |
| [dataset-3-individual-part-question-2.twbx](dataset-3-individual-part-question-2.twbx) | Seafood tissue and polymer exposure analysis |
| [dataset-3-and-5-individual-part-question-2.twbx](dataset-3-and-5-individual-part-question-2.twbx) | Combined tissue exposure and health-effect context |
| [dataset-5-individual-part-question-2.twbx](dataset-5-individual-part-question-2.twbx) | Polymer health effects and body-system analysis |

## How To Open

Open `.twbx` files in Tableau Desktop, Tableau Public, or a compatible Tableau viewer. Packaged workbooks include the workbook and embedded data needed to inspect the analysis.

## Workbook Metadata

Each workbook contains one worksheet. The files are included as editable evidence of the Tableau analysis rather than static screenshots only.
